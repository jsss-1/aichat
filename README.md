# v2：回复中

用户发送问题之后，显示回复中，得到回复后显示。

前端发送请求之后，先会得到“回复中”；
之后，去轮询获取最新回复。

后端接受请求之后，先存入到数据库中一个未回复请求。
然后异步得到回复之后，再去更新数据库。
